ربات اسکالپ کلاسیک ۵ دقیقه‌ای Toobit/OKX

ویژگی‌ها:
- تحلیل کامل از OKX روی نمادهای SWAP
- اجرای واقعی روی Toobit Futures در صورت روشن بودن ترید
- فارسی بودن پیام‌ها، پنل، آمار و نتیجه‌ها
- آمار جدا برای نتیجه عادی OKX و نتیجه واقعی Toobit
- TP ثابت 0.70% و SL ثابت 0.45%
- نمادهای هماهنگ‌شده: OKX مثل SOL-USDT-SWAP و Toobit مثل SOL-SWAP-USDT
- اگر یک نماد خطا بدهد، فقط همان نماد رد می‌شود و ربات ادامه می‌دهد

نصب:
1) pip install -r requirements.txt
2) فایل .env.example را کپی کن و با نام .env ذخیره کن
3) توکن تلگرام، چت آیدی و کلیدهای Toobit را داخل .env بگذار
4) اجرا:
   python main.py

دستورات تلگرام:
/راهنما
/پنل
/آمار
/حذف_آمار
/موجودی
/پوزیشن
/دلار_ترید 10
/لوریج_ترید 10
/حداکثر_پوزیشن 3
/ترید_فعال
/ترید_خاموش

نکته امنیتی:
ترید واقعی به صورت پیش‌فرض خاموش است. برای روشن شدن اجرا روی Toobit باید از دستور /ترید_فعال استفاده شود.
