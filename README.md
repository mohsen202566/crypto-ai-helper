# ربات فارسی کریپتو یک‌ساعته

این نسخه بر اساس معماری توافق‌شده ساخته شده است.

## قانون‌ها

- ربات کاملاً فارسی است.
- تحلیل همیشه از اوکی‌اکس انجام می‌شود.
- معامله واقعی فقط روی توبیت انجام می‌شود.
- فایل `toobit_client.py` تغییر داده نشده است.
- سیگنال فقط دو نوع دارد: رئال و نرمال.
- رئال و نرمال از نظر تحلیل هیچ تفاوتی ندارند.
- خاموش بودن ترید یا پر بودن اسلات فقط باعث نرمال شدن سیگنال می‌شود.
- اتو سیگنال همیشه فعال می‌ماند.
- هر ارز تا بسته شدن سیگنال، فقط یک سیگنال باز دارد.
- نتیجه‌ها با ریپلای روی سیگنال ارسال می‌شوند.
- تیپی با ✅ و استاپ با ❌ اعلام می‌شود.
- سود و ضرر بعد از کم کردن کارمزد اعلام می‌شود.

## دستورات فارسی

```text
ترید روشن
ترید خاموش
دلار ترید 10
لوریج 10
حداکثر پوزیشن 3
امار
آمار
```

## اجرا

```bash
cp .env.example .env
# مقدارهای .env را کامل کنید
pip install -r requirements.txt
./start_bot.sh
```
